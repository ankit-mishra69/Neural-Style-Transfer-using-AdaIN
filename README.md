# 🎨 Neural Style Transfer using AdaIN

A PyTorch implementation of **Neural Style Transfer** using **Adaptive Instance Normalization (AdaIN)**. This project transfers the artistic style of one image while preserving the original content, enabling fast and high-quality style transfer.

## ✨ Features

- 🎨 Adaptive Instance Normalization (AdaIN)
- 🧠 Pre-trained VGG Encoder
- 🏗️ Custom Decoder Network
- 📊 Content & Style Loss Optimization
- 🚀 GPU (CUDA) Support
- 💾 Automatic Model Checkpoint Saving
- 🖼️ Stylized Image Generation
- ⚡ Learning Rate Scheduler

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Deep Learning Framework:** PyTorch
- **Computer Vision:** Torchvision
- **Image Processing:** Pillow (PIL)
- **Data Processing:** NumPy
- **Training Utilities:** tqdm
- **Optimization:** Adam Optimizer
- **Model Architecture:** VGG Encoder + AdaIN + Custom Decoder
